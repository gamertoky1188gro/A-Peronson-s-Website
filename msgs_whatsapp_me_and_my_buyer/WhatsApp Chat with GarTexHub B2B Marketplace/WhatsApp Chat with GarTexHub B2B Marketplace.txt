Gmail API vs SMTP

বিষয়	Gmail API	SMTP

Setup complexity	জটিল (OAuth লাগে)	সহজ
Control & flexibility	কম (Google rules)	বেশি
Security	বেশি	মাঝারি
Speed / performance	ভালো	নির্ভর করে server এর উপর
Bulk sending	❌ খারাপ	✔️ ভালো
Reliability	খুব stable	provider এর উপর depend
Spam risk control	Google নিজে control করে	নিজে manage করতে হয়
